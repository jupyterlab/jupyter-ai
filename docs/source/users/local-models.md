# Using Local Models with Jupyternaut

This guide outlines the recommended path for connecting and running open models locally within Jupyter AI using Ollama.

## Prerequisites

* Have [Ollama](https://ollama.com/) installed and running on your local machine.

## Setup Steps

### 1. Start Your Local Model
Open your terminal and launch your desired open model using the Ollama CLI. For example, to run Qwen 3.6:
```bash
ollama run qwen3.6
