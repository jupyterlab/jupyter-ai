# Releases

Release notes for Jupyter AI. Each page aggregates the pull requests and
contributors from the subpackages whose version floors advanced in that
release.

These pages are the source of truth for Jupyter AI release notes. They are
generated by the "Step 0: Prep release documentation" workflow (and
`scripts/generate_release_notes.py`), which computes each subpackage's PR window
from the version floors declared in `pyproject.toml`.
